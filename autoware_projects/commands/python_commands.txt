Commands to run the different Models


1. main command path

cd autoware.privately-owned-vehicles/Models/visualizations


2. commands for images:

python3 SceneSeg/image_visualization.py -p ~/autoware_projects/weights/SceneSeg/weights_sceneseg.pth -i ~/autoware_projects/images/source/highway_stuttgart.png -o ~/autoware_projects/images/output/SceneSeg/

python3 Scene3D/image_visualization.py -p ~/autoware_projects/weights/Scene3D/weights_scene3d.pth -i ~/autoware_projects/images/source/highway_stuttgart.png -o ~/autoware_projects/images/output/Scene3D/

python3 DomainSeg/image_visualization.py -p ~/autoware_projects/weights/DomainSeg/weights_domainseg.pth -i ~/autoware_projects/images/source/cones.jpg -o ~/autoware_projects/images/output/DomainSeg/

python3 EgoLanes/image_visualization.py -p ~/autoware_projects/weights/EgoLanes/weights_egolanes.pth -i ~/autoware_projects/images/source/highway_stuttgart.png -o ~/autoware_projects/images/output/EgoLanes/


3. commands for videos:

SceneSeg

python3 SceneSeg/video_visualization_preview.py -p ~/autoware_projects/weights/SceneSeg/weights_sceneseg.pth -i ~/autoware_projects/videos/source/highway_stuttgart.mp4 -o ~/autoware_projects/videos/output/SceneSeg/highway_stuttgart_out --show

python3 SceneSeg/camera_visualization_cam.py   -p ~/autoware_projects/weights/SceneSeg/weights_sceneseg.pth  -o ~/autoware_projects/videos/output/SceneSeg/highway_stuttgart_out_cam --show
